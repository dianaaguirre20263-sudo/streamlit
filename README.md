## 🚀 Descripción del Proyecto
Este aplicativo ha sido desarrollado como proyecto final para el nivel **Integrador** del curso de Análisis de Datos en **Talento Tech**. La herramienta permite visualizar y analizar el ecosistema de **Datos Abiertos en Colombia**, identificando los sectores con mayor innovación y el uso que las entidades gubernamentales dan a la información pública.

## 👥 Equipo de Desarrollo
Proyecto realizado por:
- **Grupo 16 - Talento Tech**

## 🛠️ Tecnologías Utilizadas
- **Python**: Lenguaje base para el procesamiento y lógica de datos.
- **Streamlit**: Framework principal para la interfaz de usuario interactiva.
- **Pandas**: Biblioteca esencial para la limpieza y normalización del dataset.
- **Seaborn & Matplotlib**: Implementación de gráficos estadísticos con rigor académico.
- **Plotly**: Generación de visualizaciones dinámicas y KPIs interactivos.

## 📁 Estructura del Aplicativo
El panel está organizado en secciones estratégicas para facilitar la navegación:
1. **Landing Page**: Una bienvenida visual que explica el propósito del análisis y presenta el dataset.
2. **Dashboard Ejecutivo**: Resumen de indicadores clave (KPIs) y métricas de volumen de uso.
3. **Análisis Experto**: Exploración profunda mediante visualizaciones de Seaborn para entender tendencias sectoriales.
4. **Documentación**: Ficha técnica del proyecto, metodología de limpieza y créditos del grupo.
